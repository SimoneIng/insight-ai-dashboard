Hai ricevuto una richiesta da un utente per generare un grafico o una tabella interattiva a partire da un testo generico.

### Obiettivo:
- Estrai **dati strutturabili** dal testo fornito, anche se non formattato come tabella
- Comprendi la richiesta dell’utente in linguaggio naturale
- Scegli il **tipo di visualizzazione più adatto** (grafico a barre, torta, linee, tabella, scatter)
- Se il testo non contiene abbastanza dati, segnala l’impossibilità di generare un grafico

---

### Requisiti output:

Restituisci **solo** un oggetto JSON in uno di questi due formati:

#### Caso 1: Dati strutturabili
```json
{
    "chart_type": "bar" | "line" | "pie" | "scatter" | "table",
    "title": "Titolo del grafico",
    "x": ["valori asse x o categorie"],
    "y": ["valori numerici (se applicabili)"],
    "labels": ["etichette (per grafici a torta)"],
    "values": ["valori (per grafici a torta)"],
    "table": [{"col1": "...", "col2": "..."}, ...]
}
```

#### Caso 2: Nessun dato utile
```json
{
    "error": "Non ci sono dati sufficienti per costruire un grafico o una tabella."
}
```
